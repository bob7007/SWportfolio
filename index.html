Web Portfolio

A collection of various online based projects that serve to track my progress in the field of web development.

The Good The Bad And The Ugly

Is a Blog based on movie oriented websites that are rated from good to ugly. Take a look at old websites based on classic movies like "The Lost World" or "Space Jam" and contrast them with newer releases such as "It", "Goodzilla" and "Robocop". Read, Rate and comment on what is your favorite or absolute worst.

The Fountain Fan Page

On this movie based website you can find a trailer, movie scenes, cast and artwork, which is all put together with only HTML and CSS for style. Creating my First website exposed the limitations of my abilities and the difficulty of web development but triggered my curiosity to explore this field of study.

Project Portfolio

This portfolio webpage covers various engineering type projects. It differs from the current page in that is centered on hardware rather than software, even though there is still plenty of coding involved. The difference in visual style from my very first website is notable thanks to the incredible libraries and classes that Bootstrap has to offer.

Unit Convertor

Useful resource for anyone looking to make temperature and pressure conversions on demand. The design of the convertor was inspired from the google unit convertor which I find incredibly useful and easy to use. This application uses JavaScript language to do all the conversions and error checking on the client side.

Item Check List

Simple to do list that offers add, remove and complete functionality as well as save and load capabilities for user inserted items. The design for this project was made from the ground up, and centered around functionality. JavaScript and JQuery is the backbone of the application and basic bootstrap the skeleton from which style was later added.

Temperature Convertor

Similarly, from the unit convertor, this application allows the user to select between various temperatures to convert from. The main difference is that the application operates on the server side using PHP language. Other small features were added in order to stand apart from the original site which are mainly related to format and style.

Google Forms

Vote on your favorite baby names and receive live statistics in the form of bar and pie charts which are based on other users input. The google platform is an excellent tool that allows users with any level of tech savviness to create incredible content that is also visually appealing.

Baby Names Poll

Similarly, from the google form, users are able to select a gender and input a name. The site uses PHP to run the main logic which is then processed on the server side. The data inserted from the user is compared to a database using SQL tables which hold all the other names previously inserted, and in this way statistics regarding ranking and frequency of inputted names are returned for the client to see.

Pong

Play this classic game coded entirely in JavaScript. Users can play against the computer and keep track of their score. The game is very simplistic but much faster and difficult than the original, however future updates can be developed to add more modes of gameplay.
